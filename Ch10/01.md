## إضافة remote

يساعد على حفظ المعلومات التي يتبعها الـ git بمكان آخر

**Remote**  : هو مؤشر على المخزن الموجود على الـgithub وعند عمل push و pull يتم استخدامه.

عن طريق أمر:
```c#
<git remote add new-remote <link
```
وعند استعراض remote  عن طريق
```c#
git remote -v
```
نلاحظ أنه تم انشاءه.
